tool download （工具下载）

大家好，I’m Eden
一个非常优秀的代码编辑器:
        Sublime Text 3 官网
        下载链接
一个功能少，但是编码方便的C++图形库:
        EasyX 官网
        下载链接
一个功能完备、强大的IDE:
        VS community 2017 官网
        下载链接
学习WIN32 API的参考手册：
        WIN32 API手册
由于Eden是个萌新，不会复杂的Windows编程。但是又想要编写图形、动画程序。
所以在网络上发现了EasyX这个好东西。
虽然EasyX内包含的功能较少，
但是使用EasyX中的一些函数，大大降低了代码难度，缩短了代码长度。
VS2017有许多优点，但也有它的不足之处，修改配色方案太烦，而默认配色方案又不好看。。
在这一点上做的比较好的就是Sublime Text了。
